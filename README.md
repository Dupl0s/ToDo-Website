Website mit Backend und Frontend Server starten:
cd Backend

beim ersten Mal:
npm run build

dann:
npm start (jedes Mal)

in neuem Terminal:
cd Frontend
ng serve --proxy-config proxy.conf.json

Beide Server laufen!
